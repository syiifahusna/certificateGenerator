## Localhost

To run this project on localhost:

Must have
- git
- maven
- java 17

Clone this project:

```bash
git clone https://github.com/syiifahusna/certificateGenerator.git
```

Use master branch 
```bash
git checkout master
```

Compile project
```bash
mvn compile package
```

Run project
```bash
mvn spring-boot:run
```

project run on localhost:8081
